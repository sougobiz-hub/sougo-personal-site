# プロジェクト：そうご 個人HP

## これは何
フリーランスコーチ「そうご（ブランド：自分の型ch）」の個人ホームページ。
1ページ・3事業ハブ構成。GitHub Pages（リポジトリ: sougo-personal-site）で公開。
メインファイルは **index.html**（旧 sougo_personal_hp_v2.html。GitHub Pages配信のためindex.htmlに一本化）。
ストレングスセールスLPは **lp.html**（旧 ../自分の型ch_LP.html）を同リポジトリに同梱。

## 編集の基本方針
- 日本語で指示されたら、原則 index.html を編集する（sougo_personal_hp_v2.html は旧バックアップなので編集しない）。
- 変更後は、HTMLとして壊れていないか（タグの開閉・DOCTYPE）を必ず確認する。
- 新しくファイルを作る場合、ファイル名は必ず半角英数（日本語や全角の「ＨＰ」等は読み込み不具合の原因）。
- 公開後の変更は commit → push（GitHub Pagesは自動再デプロイ）。
